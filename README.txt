Hello, this is simple HTTP status checker

to run in linux shell:
  download to root directory
  find 'mailer.rb' file and fill in the fields
      YOUR_EMAIL =          '...'
      YOUR_EMAIL_PASSWORD = '...'
  print in shell: cd checker
  print in shell: ruby server_control.rb start -t
  watch the process working in console
  get the information on email if status is not 200
  to stop the process click 'CTRL+C'
